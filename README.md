# akan-name-generator
This is a name genarator that gives names to the congolese and those who live in the ivory coast as it holds cultural significance and and spiritual connection to the date of birth. 
## Description
A simple website that calculates the day of the week a user was born and assigns them an Akan name based on their gender. Akan names are derived from Ghanaian and Ivorian culture.

This project demonstrates programming concepts including:
- Working with data types
- Using arrays
- Writing functions
- Implementing control flow and operators
- Handling user input and validation
